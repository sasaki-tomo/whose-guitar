## サイト概要
ギター専用の投稿サイトを作り皆んなで褒め合い鑑賞することが目的です。

### サイトテーマ
自分の所有しているギターの投稿サイト

### テーマを選んだ理由
ツイッター等のsnsでは自分の所持しているギターを投稿しても新しい情報がたくさんあるため流れてしまったり、
youtube等で演奏はしているが使用機材等を知りたくても記されていることが少ないため。
### ターゲットユーザ
ギターを持っている全ての人

### 主な利用シーン
新しくギターを購入したり自分のギターを自慢したいとき
